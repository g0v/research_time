research_time
=============

碩博研究/畢業時間統計
為了讓學生在選擇教授的時候有更多的資訊
將統計

###流程
=============
先去[臺灣聯合大學博碩士論文系統](http://etd.lib.nctu.edu.tw)的論文瀏覽撈資料
像是學生入學時間、畢業時間、教授姓名、學號、學校、科系…
但這系統畢業時間只有年份

不過各學校圖書館系統會有更詳細的時間
像是[交大圖書館](http://webpac.lib.nctu.edu.tw/)
```bash
畢業學年度: 97 
校內電子全文開放日期 2014.4.1 
```
可以推測他是97年的4月1號畢業的
